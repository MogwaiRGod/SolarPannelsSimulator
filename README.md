SolarPannelsSimulator
===
Simulateur en ligne d'installation de panneaux photovoltaïques.

## Technologies

|Framework|Moteur de vues|Langage back|Bibliothèque de front|
|---|---|---|---|
|Symfony 6| Twig 3.1.2| PHP 8.2|Bootstrap 5
